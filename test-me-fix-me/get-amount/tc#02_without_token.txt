Тест-кейс 2: Попытка получения значения amount без токена.

Сценарий:

    Получает данные по валютам из https://www.cbr-xml-daily.ru/latest.js
    Метод возвращает значение amount
    Возвращает в ответе json вида:

{
 "result":"Your amount of money is 11.0 rub(0.12008576305 usd)"
}

Цель: Проверить выполнение сценария по получению значения amount без token.

Шаги:

1. Отправить запрос без токена на /get-amount.
2. Получить ответ содержащий ошибку.


Ожидаемый результат: Ответ от сервера вида:

{
  "timestamp": "04-07-2021 08:57:39",
  "status": "ERROR",
  "message": "error description"
}
