Тест-кейс 2: Проверка установки большого значения.

Цель: убедиться, что на попытку установления большого значения, ответ от приложения: статус-код 200,
значение успешно установленно.

 
Шаги: 

1. Отправить запрос на метод /set-amount, содержащий большое значение (например, 1000000).

2. Ответ: Статус-код 200 ок, значение успешно установленно.

Ожидаемый результат: Ответ от сервера вида:

{
    "result": "Your amount of money now is 1000000.0"
}
