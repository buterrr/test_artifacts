Тест-кейс 1: Успешное установление значения метдом /set-amount.

Цель: убедиться, что метод устанавливает значение, передаваемое в корректном запросе с корректным токеном.

Сценарий:

    Метод принимает число(value) на вход
    Устанавливает значение amount пользователя равное value
    Возвращает в ответе json вида:

{  
 "result":"Your amount of money is 10.0"  
} 
 
Шаги: 

1. Отправить корректный запрос на метод /set-amount.

2. Получен ответ от сервера об успешном установлении значения amount.

Ожидаемый результат: Ответ от сервера вида:

{  
 "result":"Your amount of money is 10.0"  
}
