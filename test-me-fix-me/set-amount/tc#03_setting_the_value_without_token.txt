Тест-кейс 3: Установка значения без токена.

Цель: убедиться, что на попытку установления значения без токена, от приложения приходить ответ с ошибкой 403.

 
Шаги: 

1. Отправить запрос на метод /set-amount без токена.

2. Получен ответ, содержащий ошибку 403.

Ожидаемый результат: Ответ от сервера вида:

{
    "timestamp": "2024-10-14T09:06:42.991+00:00",
    "status": 403,
    "error": "Forbidden",
    "path": "/set-amount"
}
